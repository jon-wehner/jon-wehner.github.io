OpenCarts

Live Site | Github

OpenCarts is an Express/React app for users to reserve times at popular food carts and skip the line.

  • Employed JSON Web Token and bcrypt.js to ensure secure user authentication and that passwords are not stored as plain text to provide increased security of user data.
  • Created a custom carousel component on the front end to implement lazy loading when users browse available food carts thereby displaying a dynamic and responsive user interface.
  • Implemented custom reservation algorithm to check for available time slots and allow users to make reservations based on the most up to date and current availability.
